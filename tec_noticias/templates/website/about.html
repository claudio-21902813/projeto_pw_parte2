Técnicas usadas

Técnica Descrição
HTML O HTML foi usado para construir o website - os textos,menu,botões - sendo a base da construção do projeto. Foram usados os elementos do HTML 5 para boas práticas, sem o uso de excessivo dos div's.
CSS Responsável pela estilização do website e pela responsividade; Auxiliram na estética das páginas e nas animações dos elementos
Django O Django serviu como Backend para o funcionamento do website e do ASP pois as paginas renderizam os conteúdos consoante o pedido efetuado
JavaScript O JavaScript funcionou para mudar os conteúdos usando o DOM e para carregar dados da API do backend
